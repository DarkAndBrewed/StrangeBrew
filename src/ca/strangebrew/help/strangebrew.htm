StrangeBrew Overview

StrangeBrew is a tool that lets you formulate homebrew recipes and track batches that you have brewed.

As a formulator, it helps you predict the colour, bitterness, starting and finishing gravities, and alcohol content of your beer. You can easily check to see how closely your recipe is predicted to match the style of beer you're trying to brew.

As a batch log, StrangeBrew lets you keep notes about the brewing process, so that you can replicate a recipe, or improve your brewing techniques batch by batch.

StrangeBrew does not teach you about recipe formulation, and good recipes require some knowledge of ingredients and styles. For that purpose I recommend some of the excellent brewing books available on the market: How to Brew by John Palmer and Designing Great Beers by Ray Daniels. These books, used with StrangeBrew, will help you become a better brewer, and make great beer!

What's new in this release:

Version 1.8 includes some exciting new features, lots of enhancements, and some bug fixes. For a full list of bug fixes, see the readme.rtf file that is installed in the StrangeBrew application folder.

New features

    • The Mash Manager has been improved: two new mash fields have been added to better model a mash: End Temp and Ramp Min, to represent cooling and ramping. See a colourful representation of your mash by clicking the "graph" button. Temp lost to tun is now saved with the recipe, rather than as a global option. There are two new water size calculations: "Misc losses" and a 3% wort volume shrinkage from chilling.
    • You can now set the boil temperature for your area, which can affect mash step directions for infusions.
    • With the new Dilution Tab, you can easily see the effects of diluting your beer to reach a target OG or IBU. You can add dilution information to your recipe report.
    • HTML export saves an HTML version of your recipe, which you can publish on the web or print from a browser. You can optionally select a CSS from the StrangeBrew directory to format your recipe.
    • The Refractometer Manager lets you to perform various brewing calculations with your refractometer. Convert brix to OG, figure out the percent alcohol based on a final gravity hydrometer and refractometer reading, and more.
    • Right click on a hop and enter the appropriate data to estimate how much alpha acid has been degraded over time.
    • You can now use a database anywhere on your computer, which makes it easier to automatically back up your data for security. Use the Database tab on the Preferences screen to set this option.
    • You can now view batch sparge calculations on the mash manager.
    • You can now set the sort order for recipes on the Database tab of Preferences. You can sort by the old way (newest recipes last), by date, by name, or reverse order (newest recipes first).

Several new minor features are included as well:

    • StrangeBrew now remembers your screen size between sessions.
    • There's a new hop type for plugs.
    • The evaporation factor is now stored per recipe rather than as a global constant. You can choose to calculate evaporation based on percent, or as a constant volume/hour.
    • The Conversions screen can be kept open in the background. It now supports SG to Plato conversions.
    • You can now set the default values for hop utilization factor, used in the Tinseth calculation. The number 4.15 is what Tinseth recommends, but you can set it higher or lower depending on the utilization you get in your particular brewery.

Getting started

This section contains the information you need to install StrangeBrew, start the software, and get technical support.

Installing StrangeBrew

Before you install StrangeBrew, ensure you have enough free disk space. StrangeBrew requires about 6 MB of free space.

To install StrangeBrew:

1. Unzip the StrangeBrew install file (brew18.zip or brew18upgrade.zip) with Winzip, PKzip, or other utility. You can put the unzipped install files into a temporary directory and delete them after StrangeBrew is installed.
2. Run the install file, setup.exe.
3. You can select where the software is installed, or specify a different directory. The default directory is C:\Program Files\Strangebrew\. If you are upgrading from StrangeBrew 1.7, be sure to install into the same directory as version 1.7.

Upgrading from a previous version

If you are upgrading from StrangeBrew 1.7:

1. Back up your StrangeBrew 1.7 folder, especially the database subfolder (called "Data").
2. Download the upgrade file brew18upgrade.zip, and follow the install instructions. You should install the upgrade files into your existing StrangeBrew directory.
3. After you have installed the upgrade, run the 18upgrade.exe file in the StrangeBrew folder, and click "upgrade". This file upgrades your version 1.7 database to version 1.8. The software will not run if you do not run the upgrade file first.

If you are upgrading from StrangeBrew 1.4 to 1.6:

1. Before installing the upgrade, export the recipes you would like to upgrade as xml files, and store them in a folder outside the StrangeBrew folder.
2. Uninstall StrangeBrew 1.4-1.6.
3. Download the full version of StrangeBrew 1.8 (brew18.zip) and follow the install instructions.
4. Run the program and import each recipe into the database.

Starting the software

You can start StrangeBrew one of two ways:

1. Click the Windows Start button and select Programs > StrangeBrew > Brew.
2. Double-click the StrangeBrew icon on your desktop.

Setting your preferences

If this is the first time you have run StrangeBrew you should set your preferences before you create new recipes. You can set preferences for:

    • new recipe defaults for units, recipe size, and mash specifications
    • cost and carbonation defaults
    • brewer information for competition entry forms
    • timeline defaults
    • calculations
    • screen appearance

Creating a new recipe - step by step

This section will take you step by step through formulating a new StrangeBrew recipe, and illustrate how simple the interface is to use.

All these steps use the StrangeBrew main screen.

1. When you start StrangeBrew, the Find Recipe window appears. Click New Recipe to create a new, blank recipe.
2. Enter a Recipe Name. Don't be shy, be creative!
3. Enter your name. You're the brewer!
4. Select a beer style from the Style list.
Tip: You can quickly jump to an item in any list in StrangeBrew by typing in the first letter or two.
5. Select a yeast from the Yeast list.
6. Change the Date (if you want to) - this field is automatically set with to date the recipe is formulated.
7. Edit the recipe size and the time it is boiled. You can do this one of two ways: enter the amount you want to collect before the boil, or the amount you want to put into the fermentor. Entering one value will recalculate the other, based on the evaporation rate set in preferences and the amount of time the wort is boiled. Hold the mouse over one of these edit boxes to view the evaporation rate. The default recipe size and units is set under Preferences.
8. Indicate whether this is a mashed recipe or not. If not, only extract and steeped grains will appear in the malts list. For mashed recipes, efficiency only applies to steeped and mashed grains - extracts and sugars contribute 100% of their fermentables. Only mashed grains are included in the recipe's mash schedule.
9. Select fermentables and hops. Click the + button to add a malt or hop. Choose an ingredient from the long list of available items. As a shortcut, press the first letter of the item you wish to select to jump to that letter. Malts that are mashed are indicated by M. You can delete ingredients by selecting them, and clicking the - button.
    • The default units for malts and hops are set under Preferences, as is the method used to calculate IBU.
    • Notice that as you add malts and hops your recipe is recalculated so that you can see exactly the effect adding or changing an ingredient has on your recipe.
10. Adjust settings. You can fine-tune your recipe by adjusting efficiency (%Effic), original gravity (OG), attenuation (%Atten), and final gravity (FG). Here's what happens when you adjust each of these items:
    • %Effic: calculates the amount of fermentable sugar you obtain from mashed malts, which affects the OG. Steeped and extract malts/sugars are not affected by this setting.
    • OG: calculates the measured efficiency of the recipe. Enter your measured OG here to see what your actual efficiency is.
    • %Atten: calculates the FG of the recipe, and the % alcohol. Alcohol is calculated by the difference between OG and FG.
    • FG: calculates the attenuation and alcohol. Enter your measured FG here to see what the actual attenuation and alcohol for the recipe is.
11. Comments: enter any generic comments you have about the recipe. You can enter more detailed notes under the Notes tab.

Now you're ready to use some of StrangeBrew's advanced features.

Adding Fermentables

To add a malt to your recipe, click on an empty "Malt" box in the Malts grid, and select from the list that appears. Type the first few letters of the malt to find it quickly in the list. Malts that are mashed are indicated by M.

To add more malts, click on the add button under the malts grid. To delete a malt, select the malt on the grid, and click the delete button under the malts grid.

You can enter or change values in the following fields: Amount, Units, Points, Lov, and Cost/Unit.

    • Points is the theoretical maximum yield one pound of malt has in one gallon of water. The actual yield is calculated by multiplying the points by efficiency.
    • If you change the Units, StrangeBrew calculates a new Cost/Unit based on the malt's cost/unit in the ingredient database, but does not convert the Amount. To convert the Amount to another unit, right-click on the malt and select Convert To...
    • Lov is the degrees lovibond measurement of the malt. This is used to calculate the beer's colour.
    • IBU is the bitterness rating for malt extracts and kits. This field is read-only on this screen. Enter a value in the Malts database. Some manufacturers list the IBU rating for a malt on the can or kit box. The IBU rating is IBU / pound / gallon amount. For example, a 3 pound bitter kit has a rating of 25 IBUs. It makes a 6 gallon recipe. The number you enter under IBU is (25 / 3), divided by 6, or 1.38.

Adding Hops

To add a hop to your recipe, click on an empty "Hop" box in the Hops grid, and select from the list that appears. Type the first few letters of the hop to find it quickly in the list.

To add more hops, click on the add button under the hops grid. To delete a hop, select the hop on the grid, and click the delete button under the hops grid.

You can enter or change values in the following fields: Type, Alpha, Amount, Units, Add, Min, and Cost/Unit:

    • Type can be "leaf," "pellet," or "plug". Pellet hops typically have more bittering power than whole hops or plugs; the percentage of bittering is set under Preferences on the Calculations tab. The default for pellet hops IBUs is 20% more than leaf hops IBUs.
    • Alpha is the percent by weight of alpha acid. This value is selected from the Hops database, but it may be changed for the recipe.
    • Add is the way hops are added. You can add hops to the boil (default), to the mash, during the sparge (FWH or First Wort Hopping), and to the fermenting beer (Dry or dry hopping).
    • Min is the minutes the hops are boiled with the wort. Longer boil times will yield more bitterness.
    • If you change the Units, StrangeBrew calculates a new Cost/Unit based on the hop's cost/unit in the ingredient database, but does not convert the Amount. To convert the Amount to another unit, right-click on the hop and select Convert To...

Estimating hop degradation

To estimate how much alpha acid a hop has lost, use the degredation calculator. Right-click on a hop that you've entered, and select Hop Storage from the pop-up menu.

The Hop Storage Manager shows:

    • Original hop alpha: the original alpha acid content for the hop.
    • Hop storage index: how well the hop variety keeps its alpha acid over time. Some varieties store better than others, and have a higher HSI. This number is stored in the ingredients database, but you can change it here.
    • Date bought: The date you bought the hops, or the harvest date, if known.
    • New alpha: The new calculated alpha based on the entered data.
    • Storage temp: The temperature the hops have been stored at. Hops store better at lower temperatures.
    • Storage Method: The way the hops have been stored. Oxygen free or barrier bags are the best way to store hops.

Checking style conformance

StrangeBrew makes it very easy to confirm that your recipe will match a BJCP style. To view style conformance, click the Style tab. You can adjust malts and hops while viewing this tab until your recipe meets the criteria of a BJCP style.

The Style Tab shows:

    • Style: selected BJCP style for this recipe.
    • Recipe Conformance: BJCP specifications for this style, and the recipe's current predicted specifications. Numbers in red fall outside the style's specifications.
    • Matched styles: other BJCP styles that the recipe matches.
    • Fudge Factor: move the slider to add a bit of "fudge" to the recipe's conformance. You can add up to 15% leeway on the specifications to see if your recipe is close to other styles.

Resizing / Converting Recipes

You can easily resize your recipe, scaling it up or down and converting to different units. To resize a recipe, select File > Resize Recipe...

Select the new size and units for the recipe. At the same time you can convert the malt and hops units by checking Convert Malts and Convert Hops, and selecting new units from the list.

Entering recipe notes

StrangeBrew allows you to track comments about different stages of the brewing process in the recipe's notes area. To view the notes for the current recipe, click the Notes tab. If you brew a recipe, a note is added to the notes database with the date.

To add a new note, click the + button. To delete the currently selected note, click -. Double-click a note to view a dialog with a larger editing window for the note.

You can enter:

    • Date - The current date appears here; you can change it if you wish.
    • Type - Select the type of note from the list. You can also enter a type description.
    • Note - Enter the text of the note.

Adding miscellaneous ingredients

To view miscellaneous ingredients for your recipe, click the Misc tab.

Use this tab to add items from the Miscellaneous database. Select items from the list, enter an Amount, Units, Cost, and Notes for how the ingredient was used. A description of the ingredient appears at the bottom of the main screen.

Enter the time that the ingredient is added during the recipe for the ingredient to show up in the recipe timeline. Select a Stage and a Time. For example, Irish moss is typically added during the last 20 minutes of the boil. For this ingredient, set the Stage to "boil" and enter 20 for the time.

Calculating cost

To view the cost for your recipe, click the Cost tab.

The Cost tab displays the total grain cost and total hops cost, as well as yeast and miscellaneous ingredient costs. Enter an amount for miscellaneous costs (such as propane, equipment, etc).

The default bottle size is set on the Cost and Carbonation tab under Preferences. The number of bottles required and the cost per bottle is calculated automatically.

Calculating carbonation

To view the carbonation calculations for your recipe, click the Carbonation Tab.

The Carbonation tab calculates how much priming sugar you need to attain a target volumes of CO2 in bottles, and the PSI you need to force carbonate the beer in a keg. Details of carbonation calculations can be found under "Calculating Carbonation" on page 52. You can enter the following settings:

    • Bottling Temp: the temperature the beer is bottled at (this will probably be different for ales and lagers). This determines the amount of dissolved CO2 still in the beer.
    • Serving Temp: the temperature kegged beer will be served at. This affects the PSI used to force carbonate the beer.
    • Temp Units: select the units for temperature, F for Fahrenheit and C for Celsius.
    • Target Vols CO2: the target volumes of CO2 in the served beer, both bottled and kegged. Here is a table of beer styles and target volumes of CO2:

Diluting wort

To view the effects of diluting your wort, click on the Dilution tab.

This tab shows:

    • Post Boil: the volume of wort after the boil.
    • Dilute with: the amount of water you want to dilute the wort with.
    • Total vol: the total volume of wort, after dilution.
    • Diluted IBU: the estimated bitterness, after dilution.
    • Diluted OG: the estimated gravity, after dilution.

Finding a Recipe

The Find Recipe dialog shows all the recipes in the recipe database. Click Open to open a selected recipe, or click New to create a new recipe from scratch.

The Folders list shows recipes in three categories:

    • Style - recipes that match a certain style
    • Brewer - recipes sorted by brewer
    • Folder - recipes that are in a folder

Click Filter... to view the filter fields. You can filter the list of recipes by clicking the filter field and entering information in the field. You can filter on recipe Name and recipe Style. The name filter is case sensitive, so if you capitalize your recipe names (and you should!), capitalize the name in the name field.

Duplicating a recipe

You can make an exact copy of the current recipe. For example, if you wish to brew a similiar or identical recipe to one you have previously brewed, use the duplicate recipe feature.

To make a copy of the current recipe, select File > Duplicate Recipe.

Organizing recipes with folders

You can organize your recipes by puting them in folders. Recipes are sorted by folders (as well as style and brewer) on the Find Recipe window. When you create a new recipe, it is not in a folder.

To put the recipe in an existing folder, select a folder in the Folder list.

To put the recipe in a new folder, click the < button next to the Folder list, and enter a new folder name. The current recipe is put in the new folder.

To remove a recipe from a folder, select the first blank line from the Folder list. A folder is deleted when no recipes are in it.

Locking a recipe

You can protect a recipe from unwanted changes by locking it. When a recipe is locked, it cannot be changed. You can add notes to a locked recipe.

To lock a recipe, check the Lock box. To unlock a recipe, clear the Lock box.

Conversions

The Conversions dialog allows you to convert from one unit to another. This is handy if you want to convert a recipe from gallons to litres, for example. Enter an amount and select a from and to unit. Choose either weight or volume units to convert.

Correcting hydrometer readings

You can correct the readings on your hydrometer using the hydrometer correction feature. Click on the < button next to either the OG or FG fields on the main screen to open the dialog. Enter the temperature the sample is taken at, the gravity, and the rating of the hydrometer (usually printed somewhere on the hydrometer scale in very small letters). The corrected reading will appear. Click OK to post the new reading into the OG or FG field.

Using the Mash Manager

To view the mash schedule for your recipe, select View > Mash Manager...

Add mash steps by clicking on the + button; delete mash steps by clicking on the - button. Select a step Type for a typical step temperature, or enter a step temperature under Temp.

Select a step method under Method. Note: the first step must be an infusion step.

Enter the step duration in minutes under MIN.

The mash schedule is automatically updated as you enter or change information. To see the calculations StrangeBrew uses for the mash schedule, refer to "Calculating Mash Steps" on page 48.

You can view a graph of your mash schedule by clicking Mash Graph.

    • Grain Temp: the starting temperature of the grain - usually room temperature.
    • Mash Vol: the final volume of the mash, useful to know if you're mashing in a limited space!
    • Vol Units: the units used to measure water additions, decoctions, and water use.
    • Temp Units: the temperature units you want to use, either Celsius or Fahrenheit.
    • Water: grain ratio: the qts/lb or l/kg ratio for the first infusion. A thick mash is 1 qt/lb. A thin mash is 2.5 qt/lb.
    • Total Min: Total time for the mash in minutes.
    • Grain Wgt: Total weight of grains being mashed.
    • Temp lost to mash tun: Temperature lost during mash-in. Set this number on the Cost and Carbonation tab under Preferences.
    • Sparge: The number and type of sparges you'd like to add to the mash schedule.

Water Usage

Enter the amount of wort left in the kettle (lost in hops, trub, etc) and left behind at the bottom of carboys to calculate the total water needed, and the final beer volume. The amount of water used for infusions, lost when the hot wort is chilled, and absorbed by the grain is automatically calculated. You can change the Collect and Post boil amounts as well (these are determined by the evaporation rate as well, set under preferences).

Using the Water Manager

StrangeBrew can help you emulate the water of world brewing cities easily, or to formulate a new water profile.

If you use your tap water as your primary water source, you may want to enter your own water profile in the Database Editor under Water. Municipal water reports are usually available through the city water department, or on the city's web site.

To open the Water Manager, select View > Water Manager.

    • Enter the total amount of treated water in the Total Volume Treated box.
    • If you wish StrangeBrew to calculate the optimum dilution for the water profile, check Dilute with Distilled. You can also enter an amount in the box manually. When an amount is entered or calculated in this box, this is the amount of distilled or R/O water you add to source water to achieve the Total Volume Treated amount.
    • Select the Source water and Target water. Each water's ion profile will be displayed.
    • To have StrangeBrew automatically calculate salt additions to create the target water profile, check the salts you would like to use, and then click Calc Water. To refine the salt additions, you can click Calc Water several times until the RMS number no longer changes. RMS is the "Real Mean Square" or the square root of the sum of the squares of the differences between all source and target ions (or, in simple terms, how close is the treated profile to the source?). But that's not important right now, what's important is that RMS is a small number, say under 10. In some cases it is impossible to recreate a target profile from a source profile because we don't have all the salts and acids available to mother nature.
    • If you wish to manually calculate a water profile, enter the salt amounts in each box.

Acidification:

You may wish to acidify some water, either for mashing or sparging. Enter data in the Target pH , Source pH and Source Alk boxes. Select an acid from the Acid list. The amount of acid required in milliliters per water unit is calculated. Note: this calculation is based on figures from the HBD (see the Calculations section), which may or may not be accurate. It does not calculate a number when Alk (alkalinity as CaCO3) is 0. Therefore, you should add acid in small amounts and carefully measure the pH of the water after each addition.

Salts:

    • Calcium Carbonate: commonly called chalk
    • Magnesium Sulfate: commonly called Epsom salts
    • Calcium Sulfate: commonly called gypsum.
    • Sodium Chloride: commonly called salt. You should use deionized salt; the best source is canning salt.
    • Calcium Chloride: not a common salt, but available through better homebrew shops.
    • Potassium Chloride: not a common salt, but available through better homebrew shops.
    • Sodium Bicarbonate: commonly called baking soda. Will increase pH, making water more alkaline. Good treatment for very acidic water.

Shopping list

The shopping list is a list of ingredients you require to brew the recipe. A shopping list is generated when you use the Brew Recipe command (File > Brew Recipe). Ingredients in the recipe that are in stock in the ingredients databases are not listed on the shopping list. To print a shopping list, select Reports > Shopping List.

Competition entry forms and bottle Labels

A competition entry form contains all the information you need to enter your beer in a homebrewing compeition.

Note: While the StrangeBrew entry forms and bottle labels are designed to contain all the information on entry forms for major national competitions, some competitions only accept their own entry forms. Consult with the compeition organizer before using the StrangeBrew entry form.

To print competition entry forms for a recipe, open the recipe and select Reports > Competition Entry Forms.

Enter the name of the competition under Competition Name. Select the options for the entry form, and whether you want bottle labels printed as well.

The Bottle Labels print preview appears after you have closed the Entry Form print preview.

Exporting and importing recipes

You can export your recipes from StrangeBrew to publish on the internet, and import other users' recipes into your database. You can also save your recipes as text files.

Exporting a recipe

StrangeBrew recipes are exported as xml (extensible markup language), which means they can be viewed in xml-enabled web browsers on the internet, and shared with other StrangeBrew users.

To export a recipe, select File > Export Recipe.

For more information about publishing recipes on the internet, see "Publishing StrangeBrew Recipes" on page 37.

Importing a recipe

You can import a recipe from StrangeBrew versions 1.4, 1.5, or 1.6. Recipes before 1.6 will be missing Lovibond numbers for malts. If an ingredient in the imported recipe is not in your ingredient database, it will be added. StrangeBrew shows you a list of new ingredients added to your database - you should edit the new ingredients and add any relevant details (how much you have in stock, for example).

To import a recipe, select File > Import Recipe, navigate to the recipe xml file, and click OK.

Saving a recipe as text

You can save a recipe as a text file, which is useful if you want to import into a word processing program, send to another person through email, or otherwise view your recipe as plain text.

StrangeBrew uses the Detailed Recipe Report as the template for the recipe text file, with all options printed. To save a recipe as text, select File > Save As Text.

If you wish to customize the information saved to a text file, select Reports > Detailed Recipe, and set the print options. On the Print Preview window, click the icon. Under Save As Type, select "Text File."

You can also save a simple recipe report as text. Select Reports > Simple Recipe. On the Print Preview window, click the icon. Under Save As Type, select "Text File.

Setting StrangeBrew Preferences

StrangeBrew allows you to control how certain values are calculated, and to set defaults for new recipes.

To open preferences, select Edit > Preferences.

The preferences you can set are on six tabs:

New Recipe Defaults tab

New Recipe Defaults:

    • % Efficiency: the default efficiency for a new recipe.
    • % Attenuation: the default attenuation for a new recipe.
    • % Evaporation: determines how much of the wort collected boils off (i.e. pre boil & post boil)
    • Size: default size of a new recipe, and default units of a new recipe.
    • Malt Units: default units used in a malt grid. You can change the units on the recipe per malt.
    • Hops Units: default units used in a hops grid. You can change the units on the recipe per hop.
    • Source Water: the default source water used for water treatment calculations.
    • Target Water: the default target water used for water treatment calculations.

Mash preferences:

    • Mash?: default value for whether a recipe is mashed or not. Recipes that are not mashed will filter out mashed malts, reducing the list you can choose from.
    • Mash Units: sets the volume units used to calculate infusions and decoctions. Also used to set the water/grain ratio.
    • Mash Ratio: sets the ratio of water to malt (in water volume per weight unit) for the first infusion of your mash, as seen on the mash manager. Units can be qt/lb or l/kg.
    • Temp Units: sets the default units for mash temps, either Fahrenheit or Celsius.
    • Grain Temp: sets the starting temperature of the grain, which is used to calculate the first infusion. Usually this is equal to the room temperature. Units are the same as Temp Units.
    • Mash Tun Thermal Mass: the temperature lost to the mashtun during mash-in. If you pre-heat your mashtun, set the number to 0.

Calculations tab

    • Hops Calc Method: select one of four methods for calculating hop bitterness. To learn more about the different methods, read "Calculating Hop Bitterness" on page 42.
    • Alcohol By: select by volume or by weight. To learn more about this option, read "Calculating Alcohol" on page 41 .
    • Pellet Hops IBU: sets the amount of IBU expected from pellet hops vs. whole hops.
    • Hop Calc Times: sets the way bitterness contributed by different types of hop additions is calculated. For example, setting Mash to 2 min means that Mash hops will be treated as if they were boiled for 2 minutes for estimating IBUs.
    • Water Usage: sets the defaults for wort left in the kettle, and beer left in the carboy for water usage calculations.
    • Colour Options: sets the colour measurement used for the finished beer (SRM or EBC) and whether recipe efficiency is applied to the colour calculation.

Cost and Carbonation tab

Cost defaults:

    • Misc. Cost: Sets the default for "other costs" for the recipe. Other costs associated with the recipe not accounted for by ingredient costs. This can include propane, equipment, etc.
    • Bottle Size: Sets the default bottle size for a recipe, including the units of volume for the bottles.

Carbonation defaults:

    • Bottling Temp: Temperature at which the beer is bottled. This determines how much CO2 remains in solution. The lower the temperature, the more CO2 is dissolved in the beer.
    • Serving Temp: Temperature at which the beer will be served. This determines the PSI the beer should be kegged at.
    • Temp Units: Fahrenheit or Celsius for the Bottling Temp and Serving Temp.
    • Target Vols CO2: The volumes CO2 you would like in the beer. See "Calculating carbonation" on page 13 for a table of beer styles and appropriate CO2 levels.
    • Priming Sugar: Type of sugar you would like to prime with.
    • Sugar Units: Units of weight you'd like to measure the sugar with.

Appearance tab

    • Red, Green, Blue: sets the RGB constants used by StrangeBrew to estimate the beer's colour on screen. Adjust these settings, plus your monitor's brightness and contrast (and gamma on better monitors) until the range of beer colours from Straw to Black looks correct.
    • First Screen: sets the first screen to appear when you start StrangeBrew.
    • Malt Sort Order: sets the way fermentables are sorted on the main screen and in recipe reports.

Brewer tab

This tab sets the information used to print competition entry forms and bottle labels.

Time Defaults tab

This tab sets the defaults for generating the brewday timeline.

Editing StrangeBrew Databases

StrangeBrew relies on these databases for all its ingredients and calculations:

    • The Malts Database: a list of malts, adjuncts, sugars, and fruits you can add to a recipe.
    • The Hops Database: a list of hops
    • The Yeast Database: a list of brewing yeasts
    • The Miscellaneous Ingredient Database: a list of miscellaneous ingredients, such as finings, spices, weird Papazian-esque stuff, and water treatments.
    • The Styles Database: a list of recognized BJCP styles.
    • The Units Database: a list of weight and volume measures used for conversions in StrangeBrew.
    • The Water Profiles Database: a list of water profiles for world brewing cities

Warning: be very careful about deleting items in StrangeBrew databases - once they're gone, they're gone. There is no delete button for the units database.

At any time you can print a report of all ingredients you have in stock by clicking Print Stock Report.

The Malts Database

The malts database contains the following fields:

    • Clone: click this button to create a copy of the selected malt.
    • Name: The name of the malt or fermentable, including country of origin.
    • Yield: The amount of fermentable sugar one pound of ingredient will yield in one gallon of water. This is the theoretical maximum (100%) yield - the actual yield is determined by your system's efficiency. Note that non-mashed ingredients are always treated as having 100% yield.
    • Lov: The lovibond (colour) rating of the ingredient.
    • Cost/Unit: The cost of the ingredient per Unit
    • Units: Unit of weight you would like to use with this ingredient.
    • Mash?: Indicates whether the ingredient must be mashed.
    • Steeped / Extract: Indicates whether an ingredient may be steeped or is an extract/sugar. Ingredients that are Steeped / Extract are always considered to contribute 100% yield. Ingredients that are not marked "Steeped/Extract" are filtered out of the ingredient list for non-mashed recipes.
    • IBU: the bitterness rating for malt extracts and kits. This field is read-only on this screen. Enter a value in the Malts database. Some manufacturers list the IBU rating for a malt on the can or kit box. The IBU rating is IBU / pound / gallon amount. For example, a 3 pound bitter kit has a rating of 25 IBUs. It makes a 6 gallon recipe. The number you enter under IBU is (25 / 3), divided by 6, or 1.38.
    • Amount in Stock: Indicates how much of the ingredient is in stock.
    • Description: A short description of the ingredient, including suggestions for use.
    • Show only in stock: Check this box to view only the malts that are in stock.

The Hops Database

The hops database contains the following fields:

    • Clone: click this button to create a copy of the selected hop.
    • Name: The name of the hop.
    • Alpha: The percent alpha acid by weight of the hop - determines the bittering contribution of the hop.
    • Cost/Unit: The cost per unit of the hop.
    • Units: The units of weight you would like to use with this hop (applies to the Amount field).
    • Amount in stock: Indicates the amount of the hop that is currently in stock.
    • Description: A short description of the hop and its traditional uses.
    • Show only in stock: Check this box to view only the hops that are in stock.

The Yeast Database

The Yeast database contains the following fields:

    • Clone: click this button to create a copy of the selected yeast.
    • Name: The yeast name, including manufacturer.
    • Cost/unit: The cost per use of this yeast.
    • In Stock?: Indicates whether the yeast is currently in stock.
    • Description: A description of the yeast, including it's flocculation and flavour characteristics, and preferred fermentation temperatures.
    • Show only in stock: Check this box to view only the yeasts that are in stock.

The Styles Database

The Styles database contains recognized BJCP styles. The fields for this database are:

    • Category Number: Major category number for the style.
    • Name: Name of the style.
    • Category: Category of the style.
    • OG, Alc, IBU, Lov Ranges: High and Low values for these specifications.
    • Description: A description of the style.
    • Commercial examples: Commercially available beers that exemplify the style.

The Miscellaneous Ingredient Database

The Miscellaneous ingredient database contains the following fields:

    • Clone: click this button to create a copy of the selected ingredient.
    • Name: Name of the ingredient.
    • Units: Preferred units of measure for the ingredient.
    • Cost / unit: The cost per unit.
    • Description: A description of the ingredient.
    • Show only in stock: Check this box to view only the miscellaneous items that are in stock.

The Units Database

The Units database is read-only. It will be editable in a future version of StrangeBrew so you can add your own units of measure. In the meantime, you can view the database to see the conversion factors used by the software.

    • Unit: The name of the unit.
    • = lb or gal: The conversion factor to a pound or US gallon.
    • Abrv.: Abbreviation (ironic, isn't it?)
    • View units type: Select the type of unit of measure you would like to view.

The Water Profiles Database

The Water Profiles database contains the following fields:

    • Name: Name of the city
    • Ca: Calcium ion concentration, in parts per million (ppm)
    • Mg: Magnesium ion concentration, in ppm
    • Na: Sodium ion concentration, in ppm
    • SO4: Sulfate ion concentration, in ppm
    • HCO3: Carbonate ion concentration, in ppm
    • Cl: Chloride ion concentration, in ppm
    • Hardness: water hardness in ppm
    • TDS: Total Dissolved Solids, in ppm
    • PH: pH of the water
    • Alk: Alkalinity as CaCO3 of the water
    • Description: a description of the water profile, including the published source of the water profile.

There are two buttons on this screen as well. Estimate Mash pH will estimate the current water profile's pH when used in an, and Estimate Alk will estimate the current water profile's alkalinity. Both are estimates only, and are based on the water profile's ion concentration.
