Managing Inventory

StrangeBrew allows you to track how much of each ingredient you have on hand. The StrangeBrew inventory features are:

    • Stock management in the ingredient databases;
    • An Inventory manager which shows you what you need to buy for the current recipe;
    • A shopping list which is generated automatically when you brew a recipe.

To manage your inventory, follow these steps:

1. First, enter the amount and units of each ingredient you have under Edit > Databases. For example, if you have 10 kilograms of Two Row malt, enter this information in the Malts database. You can print a total inventory report under Reports > Inventory.
2 When you have formulated a recipe and are ready to brew it, see what inventory you have on hand and what you need to buy on the Inventory Manager under View > Inventory.
3 To brew the recipe (which subtracts ingredients from the stock in the ingredients database, and creates a shopping list) click the Brew Recipe button on the Inventory Manager, or select File > Brew Recipe on the main screen. Note: an existing shopping list will be overwritten when you brew a recipe.
4 To view the shopping list, select View > Shopping List. You can add or delete items from the shopping list manually. To print the shopping list, select Reports > Shopping List.
5 Remember to update the ingredients databases when you buy new ingredients!

Publishing StrangeBrew Recipes

You can easily publish your StrangeBrew recipes on the internet, or send them to others to view, print, or import into StrangeBrew. The StrangeBrew export format is XML, or Extensible Markup Language, which means it is compatible with any XML enabled browser (MS Internet Explorer 5 or better and Netscape 6 or better). However, to view the recipe in a browser, the XML stylesheet strangebrew.xsl must be present in the same directory as the recipe. Here are the steps to follow:

1. Open the recipe you wish to publish.
2 Select File > Export Recipe. The default name is you're recipe's name, but you can change the file name.
3 FTP or transfer the exported recipe to your web site.
4 FTP or transfer the file strangebrew17.xsl to your web site, in the same directory as the recipe. You only need to include the stylesheet. Other exported recipes in this directory can share the same file.
5 Create a link to the xml file on a web page.

Importing other users' recipes

You can import the recipes other StrangeBrew users have created by downloading the xml recipe file and importing it. There are several recipes available at the StrangeBrew homepage (www.strangebrew.ca), and on various user sites. To download a recipe, follow these steps:

1. Find the recipe you'd like to import on the internet. You can view xml files with Microsoft Internet Explorer 5 or better.
2 Save the xml file to your computer. In IE the command is File > Save As... Browse to a directory you'd like to save the file in, select "Text File" under "Save as type", and enter a file name.
3 You may have to change the file's extension to .xml from .txt, depending on how your browser saves text files.
4 Open StrangeBrew, select File > Import, and browse to the saved recipe.
5 Click OK.
6 If the recipe contains malts, hops, or yeast that is not in your ingredients database, you will see a message telling you which new ingredients have been added. You should open the ingredients database and add information about these new ingredients (such as your cost, inventory, and description).

There are some limitations to importing recipes:

    • StrangeBrew 1.5 recipes imported into version 1.6 - 1.8 will be missing the lovibond numbers for malts. Simply re-select the malts in the list and the lovibond will be updated.

Backing up StrangeBrew data

I recommend that you backup your StrangeBrew database at least once per month, or more frequently depending on how often you add or edit recipes. Backing up your StrangeBrew database is easy, and will save you a lot of grief if you experience a computer or hard disk crash.

To backup your StrangeBrew database, follow these steps:

1. Navigate to your StrangeBrew data directory (typically it would be C:\Program Files\StrangeBrew\Data\).
2 Select all the files, or the entire directory.
3 Use an archiving program, such as WinZip or PKZip to archive all the files in the directory.
4 Store the archive off the computer, such as on a floppy disk or CD Rom.

To restore a backed up database, copy the archived files back to the StrangeBrew data directory.

Calculating recipes that use a mini-mash or steeped grains

From StrangeBrew's perspective, any recipe that uses grains or adjuncts that don't go directly into the kettle (steeped grains or a mini-mash) is considered a mashed recipe. For a mini-mash, check "mash" and select your ingredients. The Mash Manager will adjust to include only those ingredients that are checked "mashed" in the malts database. Mini-mashes tend to be lower in efficiency, so adjust your efficiency rating downward - 40-50% is probably a good range.

For steeped recipes, uncheck "mash" and select your ingredients. Some ingredients can be mashed or steeped (such as crystal malts). These ingredients will appear in the malts list. Grains that must be mashed will be filtered out.  Finally, adjust your efficiency downward (50% should be in the ballpark) since steeped grains contribute less fermentables than mashed grains.
